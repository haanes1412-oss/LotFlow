LotFlow помогает продавцам LZT Market переоценивать сразу много лотов. Инструмент загружает аккаунты, ищет рыночные аналоги, предлагает цены с объяснением и после проверки применяет подтверждённые изменения.

## Возможности

- загрузка собственных лотов через LZT Market API;
- поиск и сравнение рыночных аналогов;
- рекомендуемая цена, диапазон и уверенность расчёта;
- отдельные настройки для 24 категорий;
- калибровка по контрольным ценам продавца;
- ручное изменение любой рекомендации;
- очередь подтверждённых изменений;
- массовое применение цен на LZT Market;
- автозалив всех актуальных категорий из живого каталога LZT через `fast-sell`;
- параллельная очередь до трёх аккаунтов с паузой и восстановлением;
- защитная стартовая цена и применение рекомендации только после safety checks;
- демонстрационный режим без API-токена.

## Безопасный автозалив

Перед публикацией можно открыть конструктор выбранной категории и самостоятельно настроить любые параметры из живой схемы LZT: обязательность, влияние, диапазоны, правила и контрольные цены. Сохранённый профиль применяется к проверенной карточке сразу после публикации. Для Steam это позволяет учитывать хоть десятки связанных признаков вместо одной жёсткой формулы.

Категории автозалива загружаются из текущего каталога LZT Market. LotFlow принимает строки `login:password` или `login:password:email:email_password`, создаёт отдельное состояние для каждой строки и отправляет аккаунты через официальный `POST /item/fast-sell`. Общие поля формируются универсальным builder’ом, а обязательные параметры конкретного checker можно передать через безопасный объект `extra`. Для WoT и WoT Blitz уже встроены регион и формирование заголовка по проверенным данным.

После ответа сохраняется проверенный `item_id`, обновляется заголовок, загружается карточка и запускается обычный оценщик LotFlow с выбранным профилем, аналогами, историей продаж и confidence. Пароли хранятся в локальной очереди только в зашифрованном виде и удаляются после публикации. При неоднозначном timeout запрос создания не повторяется вслепую: LotFlow сначала пытается найти уже созданный лот по уникальному временно́му заголовку. Предложенная и фактически применённая цены сохраняются раздельно. Если оценка ненадёжна, лот остаётся на защитной цене со статусом «Нужна проверка».

## Запуск на Windows

1. Распакуйте архив.
2. Установите Node.js 20 или новее, если он ещё не установлен.
3. Откройте файл `start.bat`.
4. Введите токен LZT API или нажмите Enter для демонстрационного режима.
5. Откройте http://127.0.0.1:4173.
6. Не закрывайте окно запуска во время работы.

## Запуск на Linux или macOS

1. Откройте папку LotFlow в терминале.
2. Запустите `start.sh`.
3. Введите токен LZT API или нажмите Enter для демонстрационного режима.
4. Откройте http://127.0.0.1:4173.

## Безопасность

Токен LZT API передаётся приложению локально и не сохраняется в браузере. Данные аккаунтов в очереди шифруются ключом, производным от локального секрета, не возвращаются во frontend и удаляются после публикации. Не добавляйте токен, файлы окружения, логи и рабочие данные в публичный репозиторий.
